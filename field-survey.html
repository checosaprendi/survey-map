현장조사 손지도

회원가입 없이 현재 작업자를 설정해서 사용합니다.
자기 장소만 수정·삭제·드래그 이동할 수 있고, 조사유형은 기타 직접입력까지 지원합니다.

현재 작업자 설정

현재 작업자가 아직 설정되지 않았습니다.

선택한 장소 편집

먼저 현재 작업자를 설정하세요.
색상을 선택하세요

데이터 관리

업무 채팅방

현재 작업자를 설정하면 채팅에 참여할 수 있습니다.
아직 등록된 업무 메시지가 없습니다.
실시간 채팅은 Firebase Cloud Firestore에 연결되면 GitHub Pages에서도 여러 조사원이 함께 사용할 수 있습니다. 배포 전 코드 상단의 Firebase 설정값만 채우면 됩니다.

사용 방법

1. 현재 작업자를 먼저 설정합니다
2. 지도 클릭 → 확인 후 내 장소가 생성됩니다
3. 장소 목록을 클릭하면 지도에서 해당 위치가 확대되어 표시됩니다
4. 내 장소 아이콘은 삭제하지 않고 드래그로 위치를 바로 수정할 수 있습니다
5. 조사유형이 기타면 직접 입력란이 나타납니다
6. 색상은 조사원별로 1개만 고정되며 다른 조사원은 같은 색을 쓸 수 없습니다
7. 다른 조사원이 등록한 장소는 읽기 전용입니다
8. 업무 채팅방은 Firebase 설정 후 현재 작업자 이름으로 실시간 메시지를 남깁니다
아직 등록된 장소가 없습니다.
